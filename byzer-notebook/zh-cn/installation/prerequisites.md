### 安装前置条件

本章节将为您介绍 Byzer Notebook 的前置安装条件

### 推荐的操作系统

我们推荐您将 Byzer Notebook 安装在以下操作系统中：

- Red Hat Enterprise 7.x
- CentOS 6.4+ 或 7.x
- SUSE Linux 11
- Ubuntu 16+
- macOS 10+

### 依赖环境安装

1. 安装 JDK8 或以上

2. 安装元数据库 MySQL 5.7+

### 推荐的客户端配置

- CPU：2.5 GHz Intel Core i7
- 操作系统：macOS / Windows 7 或 10
- 内存：8G 或以上
- 浏览器及版本：谷歌Chrome 67.0.3396 及以上
